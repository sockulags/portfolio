SELECTED WORK

Meritvo — AI-assisted CV platform (private beta)
  * AI proposes, you approve: every AI edit goes through a review
    queue; nothing is silently rewritten.
  * Paste a job ad to get a match score, keywords, gaps and concrete
    recommendations.
  * PDF/HTML/TXT export from a shared render model — preview and
    export are always identical.
  Tech: Next.js 16, React 19, PostgreSQL, Prisma, Stripe, OpenAI, NextAuth

Pilot — Local-first AI agent that verifies its own work (open source)
  * Local-first: runs on Ollama models with no cloud — OpenAI as an
    optional backend.
  * Reproducible eval suite: capabilities are measured, not claimed.
  * Layered safety model with timeouts, tokens and loopback by default.
  Tech: Python, FastAPI, Ollama, MCP, WebSocket, React
  Code: https://github.com/sockulags/pilot

Design-Pilot — Site builder with canvas and CLI (in development)
  * Drag and drop real React components, edit props per instance,
    nest freely.
  * Build multiple pages and layouts, wire internal links, export the
    whole project — no npm dependency, the code is yours.
  * AI assistant via a local proxy — the API key never leaves the machine.
  Tech: TypeScript, React, Vite, Node CLI, pnpm workspaces, Anthropic API

Viska — Dictation anywhere on Windows, no cloud (private beta)
  * Press F9 and speak — Swedish-trained KB-Whisper runs on a local
    GPU and types wherever the cursor is, in any Windows app.
  * Live mode commits finished phrases while you are still speaking.
  * F8 polishes selected text with a local Qwen3 model — templates
    and per-app profiles. Audio never leaves the machine.
  Tech: Python, faster-whisper, KB-Whisper, llama.cpp, CUDA, PyInstaller

referat — Meeting minutes that never leave your machine (open source)
  * Captures system audio and microphone directly — works with Teams,
    Zoom and Meet without a bot joining the call.
  * Speaker diarization via a local pyannote server: who said what.
  * Choose where the AI runs — fully local via KB-Whisper and Ollama,
    or any OpenAI-compatible/Anthropic endpoint.
  Tech: Electron, React 19, TypeScript, Tailwind 4, Python, pyannote.audio
  Code: https://github.com/sockulags/referat

Smask! — The family cookbook: tested and approved recipes (private beta)
  * AI ingestion: photograph a cookbook page, paste a link or dictate
    the recipe.
  * Shopping list that merges ingredients across recipes and groups
    by store section.
  * Image pipeline with three webp variants and portion scaling right
    in the recipe view.
  Tech: Next.js 16, React 19, Tailwind 4, Supabase, Anthropic API, sharp
